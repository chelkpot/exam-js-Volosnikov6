## 🎯 Как это работает?

### Способ 1: Студент БЕЗ Node.js
1. Открывает `test-browser.html` в браузере
2. Нажимает кнопку "Запустить тесты"
3. Видит результаты сразу в браузере ✅

### Способ 2: Студент С Node.js
1. Выполняет `npm install`
2. Выполняет `npm test`
3. Видит подробные результаты в консоли ✅

### Способ 3: Автоматическая проверка GitHub
1. Студент делает `git push`
2. GitHub Actions автоматически запускается
3. Запускаются Jest тесты
4. Результаты видны в Actions и GitHub Classroom ✅
